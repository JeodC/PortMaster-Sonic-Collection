## Installation
Add your Sonic Mania `Data.rsdk` file to `ports/sonicmania`.

## Default Controls
| Button | Action |
|--|--|
|START|Pause / Accept|
|D-PAD|Move|
|LEFT ANALOG|Move|
|ABXY|Jump / Fly|
|DOWN + ABXY|Spindash|

## Notes
Plus DLC requires a custom built RSDKv5, not meant for distribution. See the BUILDING.md guide for information.
