## Installation
Add your Sonic CD `Data.rsdk` file to `ports/soniccd`. Using the Android apk is recommended.

## Default Controls
| Button | Action |
|--|--|
|START|Pause / Accept|
|D-PAD|Move|
|LEFT ANALOG|Move|
|ABXY|Jump / Fly|
|DOWN + ABXY|Spindash|

## Using mods
Open the dev menu by pressing Select and choose your mods to activate.
